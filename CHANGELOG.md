## 0.0.1

* New Features

    * Bindings for all [WebDriver endpoints](https://w3c.github.io/webdriver/webdriver-spec.html) as of 2018-04-20
    * Integration with the [Tasty](https://hackage.haskell.org/package/tasty) test framework
